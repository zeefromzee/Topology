# API Reference

This document provides detailed API documentation for all modules in the package.

## Access Structure Module

### `access_structure.generator`

Functions for enumerating and generating access structures.

### `access_structure.validator`

Functions for validating access structure properties.

### `access_structure.converter`

Functions for converting between access structures and simplicial complexes.

## Crypto Module

### `crypto.shamir`

Implementation of Shamir's secret sharing scheme.

### `crypto.share_bounds`

Information-theoretic share size bound computations.

### `crypto.ideality`

Matroid-based ideal scheme detection.

### `crypto.threat_analysis`

Combinatorial threat modeling and analysis.

## Topology Module

### `topology.complex_builder`

Simplicial complex construction and manipulation.

### `topology.homology`

Homology group computation.

### `topology.betti`

Betti number calculation.

### `topology.euler`

Euler characteristic computation.

### `topology.vertex_analysis`

Vertex link and criticality analysis.

### `topology.visualization`

Complex rendering and diagram generation.

## Pipeline Module

### `pipeline.runner`

End-to-end pipeline orchestration.

### `pipeline.correlation`

Topological-cryptographic correlation analysis.

### `pipeline.report`

Multi-format report generation.

## Utils Module

### `utils.sets`

Set operation utilities.

### `utils.formatting`

Output formatting helpers.
